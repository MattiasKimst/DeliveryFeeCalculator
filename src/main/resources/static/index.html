Welcome to Delivery Fee Calculator

This application calculates delivery fees for food couriers based on city, weather conditions and vehicle type.

How to Run

  1. To make a GET request, replace <city> and <vehicle> with one of the cities from valid input space {Tallinn, Tartu, Pärnu} and a vehicle from the list {Car, Scooter, Bike}. You can test invalid input as well, which results in a response containing corresponding error message and null value for fee.

    URL: "http://localhost:8080/calculateDeliveryFee?stationName=<city>&vehicleType=<vehicle>"

  2. To make a GET request with optional datetime parameter, replace <city>, <vehicle> and < yyyy-MM-dd'T'HH:mm:ss > with one of the cities from valid input space {Tallinn, Tartu, Pärnu}, a vehicle from the list {Car, Scooter, Bike} and a date in provided format like 2024-03-09T10:00:00. In case of invalid date format, corresponding error message will be provided in response

    URL: "http://localhost:8080/calculateDeliveryFee?stationName=<city>&vehicleType=<vehicle>&datetime=<yyyy-MM-dd'T'HH:mm:ss>"

  3. The H2 database console could be accessed at http://localhost:8080/h2-console. Login credentials are in application.properties file.
